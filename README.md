# Triple Take

A cooperative word-guessing party game based on Fishbowl/Monikers.

## About

Triple Take is a web-based party game where players guess words through three rounds with increasingly restrictive clues. Pass a single device between players for in-person fun!

- **Round 1**: Describe with any words/gestures
- **Round 2**: One word only
- **Round 3**: Charades

## Development

This project uses Vite + TypeScript with vanilla JavaScript (no framework).

### Setup

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```
